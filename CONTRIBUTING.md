# Contributing

C# edition rules:

1. Use Godot 4 .NET APIs and C# idioms.
2. Use `.cs` examples and `csharp` code fences.
3. Do not add new GDScript implementation snippets.
4. Convert GDScript syntax to C# syntax before committing.
5. Mention validation status when a sample has not been compiled in Godot .NET.
6. Keep existing skill names stable so agent routing does not break.
